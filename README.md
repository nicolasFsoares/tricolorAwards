🏆 Tricolor Awards 🏆
---

**The virtual trophy room of Grêmio Foot-Ball Porto Alegrense!**

Here you'll find the major titles and achievements of the biggest club in southern Brazil, with a short story behind each conquest, along with the standout players who made history for Tricolor Gaúcho.

*Dedicated to every diehard supporter of this club.*

## Tech Stack

HTML, CSS, Bootstrap

## Getting Started

This is a static site with no dependencies or build step — just clone and open it in your browser.

```bash
git clone https://github.com/nicolasfsoares/tricolorAwards.git
cd tricolorAwards
```

Then simply open `index.html` in your browser, or serve it locally:

```bash
python -m http.server
```

and visit `http://localhost:8000`.
